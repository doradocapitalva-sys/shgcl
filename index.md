# Spiral Horn OS

Internal financial-reporting and reconciliation tool for **Spiral Horn General Contractors LLC**. The application reads accounting data from QuickBooks Online and project data from JobTread, builds a normalized intermediate data layer, and surfaces operating-truth dashboards for authorized Spiral Horn personnel.

The application is not distributed externally and is not offered to third parties. It runs locally on the authorized user's computer; data accessed by the application is never transmitted to a remote server other than the source services themselves (Intuit's QuickBooks Online API and JobTread's Pave API).

This page exists to satisfy Intuit Developer's host-domain checklist for the Spiral Horn OS QuickBooks Online integration.
